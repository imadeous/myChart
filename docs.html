Introduction

MyChart.js is a modern, lightweight JavaScript charting library for building beautiful, interactive dashboards with zero dependencies. It supports bar, line, pie, mixed, and stacked charts, and is fully customizable via JavaScript.

  • Zero dependencies, pure JS/Canvas
  • Responsive and mobile-friendly
  • Customizable tooltips, legends, and colors
  • Supports multi-series, mixed, and stacked charts

Getting Started

  1. Download myChart.js and include it in your HTML:
<script src="myChart.js"></script>
  1. Add a <canvas> element to your page:
<canvas id="myChart" width="600" height="400"></canvas>
  1. Call drawChart with your data:
drawChart('bar', ['A', 'B', 'C'], [10, 20, 30], 'myChart');

Usage

You can create different chart types by changing the type argument:

  • Bar: 'bar'
  • Line: 'line'
  • Pie: 'pie'
  • Mixed: 'mixed' (bar + line)
  • Stacked Bar: 'stacked-bar'

Multi-series charts use an array of dataset objects:

drawChart('bar', ['A', 'B', 'C'], [
  { label: 'Series 1', data: [10, 20, 30] },
  { label: 'Series 2', data: [15, 10, 25] }
], 'myChart');

API Reference

drawChart(type, labels, data, canvasId, options?)

  • type: 'bar' | 'line' | 'pie' | 'mixed' | 'stacked-bar'
  • labels: string[] (X-axis or pie labels)
  • data: number[] or Dataset[]
  • canvasId: string (id of your <canvas>)
  • options: object (optional, e.g. { yAxisLabel: 'USD ($)', stacked: true })

Dataset Object

{
  label: 'Series Name',
  data: [10, 20, 30],
  borderColor: '#3b82f6',
  backgroundColor: '#3b82f6',
  pointColor: '#3b82f6',
  type: 'bar' | 'line'
}

Examples

Bar Chart

drawChart('bar', ['A', 'B', 'C'], [10, 20, 30], 'myChart');

Line Chart

drawChart('line', ['Jan', 'Feb', 'Mar'], [5, 15, 10], 'myChart');

Pie Chart

drawChart('pie', ['Income', 'Expense'], [73528.67, 85039.19], 'myChart');

Multi-Series Bar

drawChart('bar', ['A', 'B', 'C'], [
  { label: 'S1', data: [10, 20, 30] },
  { label: 'S2', data: [15, 10, 25] }
], 'myChart');

Mixed Chart

drawChart('mixed', ['Q1', 'Q2', 'Q3', 'Q4'], [
  { label: 'Bar', data: [10, 20, 30, 40], type: 'bar' },
  { label: 'Line', data: [5, 15, 10, 25], type: 'line' }
], 'myChart');

FAQ

Can I use MyChart.js with frameworks like React or Vue?

Yes! MyChart.js is framework-agnostic. Just use a <canvas> and call drawChart in your component lifecycle.

Does it support tooltips and legends?

Yes, tooltips and legends are built-in and customizable via CSS.

How do I export a chart as PNG?

Call chartInstance.exportPNG() or use the built-in download button if available.
